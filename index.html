Frequently Asked Questions

Everything you need to know about GHOST PASS

Is GHOST PASS really free and open source?

Yes! GHOST PASS is completely free and open source under the MIT license. You can view, modify, and distribute the code freely.

How does GHOST PASS protect my privacy?

GHOST PASS routes all your internet traffic through the TOR network with multi-layer encryption, preventing IP leaks, DNS leaks, and ensuring complete anonymity.

Which platforms are supported?

GHOST PASS works on Windows, Linux, and macOS. We provide both Python packages and standalone executables for easy installation.

Do you log any user data?

No. GHOST PASS has zero logging policy - no telemetry, no history tracking, no user data collection. Your privacy is our priority.

How do I get started?

Clone the repository, install dependencies, configure TOR, and run python -m ghostpass. Check our GitHub for detailed installation instructions.
